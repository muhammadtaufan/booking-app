# Booking App
Simple Rails API to create reservation from various third parties payload request

## Getting Started
### Prerequisites
```sh
Ruby version 3.0.0
Rails version 7.0.4
```

### Installing

- Clone the repo
```sh
git clone https://github.com/muhammadtaufan/booking-app.git
```

- Install dependencies
```sh
bundle install
```

### Usage
#### Run the server

```sh
rails s
```
